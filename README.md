# Banking Fraud Detection System

## Overview

This project detects suspicious banking transactions using data analytics and visualization techniques.

## Features

* Generate synthetic transaction data using Python
* Clean and preprocess transaction data
* Analyze fraud patterns using SQL
* Visualize fraud insights using Streamlit Dashboard
* Manage project using Git and GitHub

## Tech Stack

* Python
* Pandas
* SQL
* Streamlit
* Git & GitHub

## Project Structure

* data_generator/ → Generate transaction data
* pipeline/ → Data cleaning and preprocessing
* analytics/ → SQL fraud analysis queries
* dashboard/ → Streamlit dashboard

## Results

* Total Transactions: 1000
* Fraud Transactions: 45
* Highest Fraud: UPI
* Lowest Fraud: Net Banking

## How to Run

python3 data_generator/fake_transactions.py

python3 pipeline/clean_transactions.py

streamlit run dashboard/dashboard.py
